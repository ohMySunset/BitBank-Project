# BitBank-Project
Eclipse를 활용한 스마트뱅킹 구현 미니 프로젝트

* 자바 기초 내용을 토대로 간단하게 콘솔에 구현한 은행업무 기능🏦
* 주요 기능 : 로그인, 회원가입, 계좌개설, 입금, 출금, 계좌이체, 거래내역 조회, 대출 심사, 대출금 상환, 대출내역 조회
